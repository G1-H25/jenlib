Detailed Description

Sensor to Broker measurement payload.

Temperature is encoded in centi-degrees Celsius (e.g., 2312 => 23.12 C). Humidity is encoded in basis points (0..10000 => 0%..100.00%). The "offset_ms" is the elapsed time since the start message was accepted.
