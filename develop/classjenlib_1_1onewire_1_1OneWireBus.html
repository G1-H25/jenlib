Detailed Description

OneWire bus interface abstraction.

Provides a minimal, platform-agnostic API for 1-Wire transactions. This is a concrete class that can be easily understood by students. All multi-byte data uses little-endian byte order as sent on the wire (LSB first) consistent with 1-Wire protocol.
