Detailed Description

template<typename Tag>
class jenlib::gpio::TypedPin< Tag >

Type-safe wrapper for pins used with external libraries.

This wrapper provides type safety while allowing conversion to raw pin numbers when needed for compatibility with existing libraries like OneWire, SPI, I2C, etc.
