Detailed Description

Protocol-level definitions for BLE sensor/broker communication. Copyright 2025 Jennifer Gott.

Defines roles, opcodes, versioning, and basic wire constraints for the jenlib BLE protocol. Transport specifics (GATT UUIDs) live in GattProfile.h, while payload layouts live in Messages.h.
